Sobre o Sistema

SG-Veiculos

Projeto que simula um sistema de Gestão de Veículos e Proprietarios com operações de Busca, Cadastro Atualização e Exclusão de Dados.
Tambem disponibilizo uma Api Rest para consumo, link no próprio Sistema

Tecnologias Usadas:

Java 8, Spring Boot, Spring Security, Spring Data-JPA, Template Engine Thymeleaf e Banco de Dados PostgreSQL
